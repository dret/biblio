TeX2Unicode Character Conversion Table

This is the character conversion table used for conversions between TeX characters and Unicode characters. If you have any comments, corrections, or additions regarding this table, please let us know. In addition to this HTML table, you may also be interested in the following resources:

  • If you are interested in a machine-readable version of the character conversion table, you can get the XML version of the table. The XML version of the table is a trivial mapping table with a very small DTD in its internal subset, so it should be easy to understand and use.
  • A sed script for converting Unicode to TeX characters. This text file is an UTF-8 file, so if your sed does not support UTF-8, you are out of luck.
  • Three XSLT 2.0 character maps for mapping Unicode characters to the corresponding TeX codes. The tables are split (the character map names are US-ASCII, ISO-8859-1, and all) so that the first two tables can be used to only map characters that are not part of US-ASCII respectively ISO-8859-1, while the other table maps all entries of the tex2unicode mapping table (hopefully producing a US-ASCII file).
